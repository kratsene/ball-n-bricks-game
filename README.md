- **Dynamic Gameplay** – Classic brick-breaking mechanics with modern twists

## 🎯 How to Play

1. **Launch** – Tap and drag to aim your ball
2. **Aim Carefully** – Position for maximum brick destruction
3. **Break Bricks** – Destroy all bricks to advance to the next level
4. **Survive** – Don't let the ball fall off the bottom of the screen
5. **Collect Power-ups** – Grab special items for advantages
6. **Achieve High Scores** – Beat your personal best

## 💻 Installation from Source

### Prerequisites

- Flutter SDK (v3.0 or higher)
- Dart SDK
- Android Studio / Xcode (for native development)

### Clone & Setup

```bash
git clone https://github.com/kratsene/ball-n-brick.git
cd ball-n-brick

# Install dependencies
flutter pub get

# Run the app
flutter run
```

### Build for Release

**Android:**
```bash
flutter build apk --release
```

**iOS:**
```bash
flutter build ios --release
```
